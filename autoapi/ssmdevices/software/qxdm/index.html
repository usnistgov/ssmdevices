configure(config_path, min_acquisition_time=None)[source]¶

Load the QXDM .dmc configuration file at the specified path, with adjustments that disable special file output modes like autosave, quicksave, and automatic segmenting based on time and file size.

save(path=None, saveNm=None)[source]¶

Stop the run and save the data in a file at the specified path. If path is None, autogenerate with self.cache_path and self.data_filename.

This method is threadsafe.

Returns:

The absolute path to the data file

_set_com_port(com_port: int)[source]¶

connects to the handset at the specified serial port.

Blocks until QXDM confirms the phone is connected.

Parameters:

com_port (int) – serial port number, COMn; 0 or None to disable

Raises:

  • TimeoutError – after waiting self.connection_timeout for connection

  • IOError – when the QXDM COM API returns code -1
